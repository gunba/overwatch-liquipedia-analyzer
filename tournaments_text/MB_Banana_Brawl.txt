==About==

===Format===
*The Banana Brawl is an open registration single elimination tournament, which evolves into a double elimination tournament once top 16 is reached.
*Match Procedure:
**'''First to 2''':during the single elimination stage;
***Map order: [[Control]] → [[Hybrid]] → [[Escort]] → start over.
**'''First to 3''':during the double elimination stage;
***Map order: [[Control]] → [[Hybrid]] → [[Escort]] → [[Assault]] → start over.
**'''First to 4''':during the Grand Finals.
*The team in the Grand Finals which did not drop to the Losers Bracket will start Grand Finals with a 1-0 score.
