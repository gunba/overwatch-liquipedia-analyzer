==About==
The Widowmaker Showdown is an unofficial solo tournament on Overwatch, where players play using only Widowmaker and only headshots.

==Format==

This tournament uses specific workshop mods for each phase (groups/playoffs). Any game that is not played using the mod of the corresponding phase will need to be replayed.
